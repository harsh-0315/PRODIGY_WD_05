# PRODIGY_WD_05

The Weather Web App is a simple and interactive application designed to provide real-time weather information for any city worldwide. It uses a clean interface, smooth interactions, and integrates with a live weather API to deliver accurate updates on temperature, humidity, wind speed, and overall weather conditions.

Users can search for a location manually or allow the app to automatically detect their current position using geolocation. The app then fetches live data such as temperature, weather description, and weather icons, making it easy to understand at a glance. It is built using HTML, CSS, and JavaScript, with API integration to deliver seamless user experience.
